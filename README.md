# Female Secondary School Completion Risk Predictor
### SDG 4 – Quality Education | PES University | Sanjana M | PES1PG25CA190

## About
This project predicts whether an Indian district is at HIGH or LOW risk 
of having low female secondary school completion rates using supervised 
machine learning.

## Folder Structure
female-education-predictor/
├── app.py                  # Streamlit web application
├── model.pkl               # Trained Logistic Regression model
├── requirements.txt        # Required libraries
└── README.md               # Project documentation
## How to Run on Your Laptop
### Step 1: Clone the repository
git clone https://github.com/sanj004/female-education-predictor
cd female-education-predictor

### Step 2: Install required libraries
pip install -r requirements.txt

### Step 3: Run the app
### Step 4: Open in browser
The app will automatically open at http://localhost:8501
